# Payment Tracker (Flat File Edition)

A Java desktop application for tracking payments, expenses, and income using flat file storage (CSV files).

## Features

- ✅ **No Database Required** - Uses CSV files for storage
- ✅ **Portable** - Copy and run anywhere with Java
- ✅ **Multi-currency Support** - Auto-converts to CHF
- ✅ **Recurrence Tracking** - Daily, weekly, monthly, yearly
- ✅ **Location & Category Management** - Add on the fly
- ✅ **Data Filtering** - By date, type, location
- ✅ **Auto-backup** - Automatic backup system
- ✅ **Export Ready** - CSV files can open in Excel

## Requirements

- Java JDK 11 or higher
- Maven 3.6+ (for building)
- VS Code (for development, optional)
